# Actividad 2.5 - Node.js

## Descripción

Esta actividad fue desarrollada para la asignatura Programación Web.

Incluye ejemplos de:

- Variables y tipos de datos.
- Condicionales.
- Ciclos.
- Funciones.
- Módulos.
- Uso del módulo `fs` para crear un archivo de texto.

## Requisitos

- Node.js instalado.

## Ejecución

Abrir la terminal en la carpeta del proyecto y ejecutar:

```bash
npm start
```

Al ejecutar el programa se mostrará la salida por consola y se generará automáticamente el archivo `resultados.txt`.
